 ## git 명령어 요약

- clone : 원격 저장소 복사
- add : 스테이지 영역에 작업 파일 추가
- commit : 스테이지 영역의 파일을 가지고 커밋(세이브)를 만든다
- push : 원격저장소에 커밋을 업로드
